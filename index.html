🏊 Swim Meet Ribbon Sticker Maker

Type your meet info, adjust counts if needed, then Print onto Avery 5160 label sheets (30 per page).

▶ How to use this tool

This tool builds a full set of ribbon labels for a swim meet automatically. Here's how the labels get created:

  • All 31 events are built in (Event #0–#30) with their names already filled in — you never retype them.
  • Type the meet date and meet name once at the top. Those two lines (“Date:” and “Meet:”) print on every label automatically.
  • Each event gets a label for every placement — 1st 2nd 3rd 4th 5th and then Participant (covers 6th place and beyond).
  • “Per place” = how many labels for each of 1st–5th. Relays default to 4 (one per swimmer); individual events default to 1. “Participant” is a separate count for the leftover swimmers.
  • Turnout changes each meet — just edit the numbers in the table for any event. Set a count to 0 to skip that placement entirely.
  • Event #0 is special — a stack of blank “8 & Under Medley Relay” labels with no placement line, used as fill-in extras.
  • Watch the Live Preview on the right — it updates the moment you change anything, so you always see what will print. Click it to enlarge.
  • Everything saves automatically. Next meet, just change the date and tweak a few counts. (Use Reset to defaults to start over.)

How to print correctly (read once)
  1. Load Avery 5160 label sheets in your printer.
  2. Click Print labels. Choose paper size US Letter.
  3. Set scale to 100% / “Actual size” — not “Fit to page,” or the labels won't line up.
  4. Set margins to None / Default; turn off headers & footers.
  5. First time: print one page on plain paper and hold it over a real label sheet to check alignment.
