# 🏊 Swim Meet Ribbon Sticker Maker

A free, single-file browser tool for printing ribbon labels for a swim meet onto
**Avery 5160** label sheets (30 labels per page).

**▶ Use it here:** https://nleaf.github.io/swim-meet-labels/

No install, no login, nothing to download — it runs entirely in your browser, and
your settings are saved automatically on the computer you use.

## How it works

1. **Type the meet date and meet name** at the top. They print on every label.
2. **Adjust the label counts** for any event (turnout changes each meet). Each event
   is pre-filled with sensible defaults; set a count to `0` to skip a placement.
3. **Click "Print labels"** and load Avery 5160 sheets.

All 31 events (#0–#30) are built in with their names. Each event prints labels for
**1st–5th Place + Participant**. Relays default to 4 labels per place (one per swimmer);
individual events default to 1. Event #0 is a stack of blank fill-in labels.

## Printing tips

- Paper size: **US Letter**
- Scale: **100% / "Actual size"** — *not* "Fit to page," or the labels won't line up
- Margins: **None / Default**; turn off headers & footers
- First time: print one page on plain paper and hold it over a real label sheet to
  check alignment.

## Notes

- The on-screen preview guides (dashed boxes, alternating colors, event markers) are
  aids only — **they never print**.
- Works offline; the only thing needing internet is the Inter font (it falls back to a
  standard font if you're offline).
